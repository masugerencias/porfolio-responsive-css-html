INSTITUTO DE FORMACIÓN EMPRESARIAL DE LA CÁMARA DE COMERCIO DE MADRID

Tuve la oportunidad de poder cursar una formación online de FullStack de la Cámara de Comercio de Madrid mientras estaba cursando la Formación Profesional de DAM. En esta formación de 250 horas aprendí el concepto y la parte fundamental de las dos grandes partes de una aplicación FullStack: FrontEnd y Backend.
El FrontEnd constaba de HTML y CSS para ofrecer la parte visual del Backend con JAVA SE y su Ecosistema SPRING.
He aquí las tecnologías usadas:

HTML CSS JAVA SE Ecositema SPRING JDBC JPA MVC REST

La última parte de esta formación constitía en el desarrollo Web para Comercio Electrónico, es decir, la creación de páginas web de tienda virtual a través de Shopify.
