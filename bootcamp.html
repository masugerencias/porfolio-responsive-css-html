¿QUÉ ES UN BOOTCAMP?

El BootCamp de Full Stack Developer es un curso/master intensivo de 4 meses donde se está continuamente practicando con las tecnologías que se utilizan en las empresas, y está orientada a la empleabilidad, haciendo de vehículo conductor de lo aprendido con los futuros trabajos.
Se realizan varios proyectos para la construcción de app web, donde las dos últimas semanas nos juntamos con las otras verticales del curso (Marketing, UX/UI, Data, Ciber Seguridad) para crear un MVP de un producto, que una empresa asociada a The Bridge propone como reto.
Estas son las tecnologías usadas para el FrontEnd (desliza hacia la derecha):

Javascript HTML CSS DOM Web Storage FlexBox Grid Bootstrap Saas Cookies PUG EJS ReactJS

Hasta aquí solo son aplicaciones que si cuentan con una gran intaracción con el usuario pero no se comunican a ninguna base de datos, por ello, hemos trabajado en la parte de BackEnd con las siguientes herramientas:

SQL PL-SQL MongoDB FireBase NodeJS Mongoose Sequelize Express JSDOC JWT Selenium MERN

Incluyendo la parte Back-end a las páginas web tenemos como resultado aplicaciones 100% funcionales, conectadas a APIs, BBDD tanto relaciones como no relacionales.
